# Simple Blog Website using TypeScript, React, NextJs, & Supabase

Developed by **Dan Louis M. Dela Cruz**.

Live site: https://withcenter-blog.danlouis.dev/

Created as an Assessment Project for Withcenter Inc Korea.

## Features

- User account authentication using Supabase Auth
- Read, Create, Update, and Delete blogs
- Uses Supabase as primary database
- Image uploads in blog posts and comments using Supabase Storage
- Utilizes TipTap rich-text editor for blog composition.
- State management handled by Redux
